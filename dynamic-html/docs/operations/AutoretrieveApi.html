adminAutoretrieveInitPost

This endpoint registers a new autoretrieve server

URL

https://api.estuary.tech/admin/autoretrieve/init

HTTP Method

POST

Response Type

String

Parameters

  • Form: addresses String

    Autoretrieve's comma-separated list of addresses

  • Form: pubKey String

    Autoretrieve's public key

GET

Response Type

String

Parameters

String
